# Customer Segmentation using RFM Analysis and Clustering

## 📌 Project Overview

This project analyzes customer purchasing behavior using transaction data from an online retail store. The goal is to segment customers using RFM (Recency, Frequency, Monetary) analysis and clustering techniques.

---

## 📊 Dataset

* Source: Kaggle
* Dataset: Online Retail Dataset
* Link: [https://www.kaggle.com/datasets/ishanshrivastava/tata-online-retail-dataset](https://www.kaggle.com/datasets/ishanshrivastava28/tata-online-retail-dataset)

---

## 🧹 Data Cleaning

* Removed missing CustomerID
* Removed negative Quantity (returns)
* Removed zero/negative UnitPrice
* Converted InvoiceDate to datetime
* Created TotalPrice
